# photo-resize-js
写真画像をクライアントサイドでリサイズして、アップロードできるようにするJavaScriptライブラリ。

画像の拡大、縮小にはCanvasを利用して、piexifを使って、EXIFも対応させる。EXIFは残す。

# 機能
- Fileで読み込む画像を渡すか、文字列で渡す
- 横方向の最大サイズと横方向の最大サイズを指定
- サムネイルを作成し直す
